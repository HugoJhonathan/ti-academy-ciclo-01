Exercicio 28

1. Como é possível descobrir qual é a fonte padrão de renderização do navegador?
2. É verdade que quando alteramos um CSS diretametne no DEV TOOL
automaticamente alteramos o arquivo CSS? 


Quest 1- R: Para descobrir a fonte padrão de utilização do navegador
basta abrir o DevTools (apertando f12), e acessar a aba "Computed".

Quest 2- R: Não, quando fazemos qualquer alteração no CSS pelo DevTools
não altera nada no arquivo original, apenas altera no memento, e quando
atualizamos a pagina toda alteração é perdida.
